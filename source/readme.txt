This folder includes the OTAWA framework source of the modules OTAWA, GLISS2, GEL, ELM, oRange, FrontC, ... as well as the MKFFX and OSWA tool.
	
Required Packages:
build-essentials
GNU g++ compiler
GNU make
flex
bison
cmake
libxml2
libxslt (libxslt1.1 and libxslt1-dev)
OCAML

Install:
$ sudo apt-get install flex bison cmake libxml2 libxslt1.1 libxslt1-dev build-essential ocaml
$ sudo make otawa-install
