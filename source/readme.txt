This folder includes the OTAWA framework source of the modules ARMv7-Loader, GLISS2, GEL, ELM, oRange, FrontC, lp solve and, finally, OTAWA containing the MKFFX and OSWA tool.
	
Required Packages:
build-essentials
GNU C++ compiler
GNU make
flex
bison
cmake
libxml2
libxslt (libxslt1.1 and libxslt1-dev)
OCAML

Install:
$ sudo apt-get install flex bison cmake libxml2 libxslt1.1 libxslt1-dev build-essential ocaml
$ sudo make otawa-install
